**Задача:** Native Request Validation (Валидация данных).

Решил придерживаться принципа минимизации зависимостей.
Написал собственные функции-защитники (`isValidTopic`, `isRecord`). Это позволяет TypeScript'у "понять" типы данных в рантайме.
Валидатор не просто проверяет, что `topic` — это строка, он проверяет наличие этой строки в массиве `TOPICS`. Это защищает нас от Prompt Injection и галлюцинаций AI.
Контроллер стал чище. Вся грязная работа по проверке данных вынесена в утилиту `validateChatRequest`.

**Итог:** API защищено. Невалидные запросы отсекаются мгновенно, не нагружая сервис AI.
